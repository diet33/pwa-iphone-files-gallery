# 파일갤러리 PWA

아이폰 파일 앱의 이미지를 사진 앱으로 옵기지 않고 바로 보는 PWA

## 기능
- 파일 앱 이미지를 사진 앱 앱범으로 옵기지 않고 보기
- 여러 장 선택 후 좌우 스와이프
- 피언쳠 확대/축소
- 더블탭 확대/원복
- 씀네일 목록
- PWA 설치 후 오프라인 실행

## 중요
- iPhone Safari는 파일 앱의 index.html을 직접 PWA로 설치할 수 없습니다.
- PWA 설치는 Safari에서 HTTPS 주소로 한 번 접속한 뒤 "공유 > 홈 화면에 추가"로 해야 합니다.
- 설치 후에는 앱 껍데기 자체는 오프라인으로 열립니다.
- 사진 파일은 보안상 매번 사용자가 직접 선택해야 합니다. iOS가 파일 앱 폴더 전체 자동 접근을 막습니다.

## 설치 및 사용 방법
1. Vercel에서 배포된 HTTPS 주소로 접속합니다.
2. Safari에서 공유 버튼 → 홈 화면에 추가
3. 홈 화면에 생성된 아이콘으로 실행
